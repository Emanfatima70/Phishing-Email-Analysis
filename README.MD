\# 🔍 Phishing Email Analysis Project



\## 📌 Overview

This project is a beginner SOC Analyst simulation that detects phishing emails using Python and basic email indicators.



\## ⚙️ Features

\- Suspicious sender detection

\- Phishing keyword analysis

\- Unsafe link detection

\- Automated email scanning script



\## 🧪 Tools Used

\- Python

\- Git \& GitHub

\- Command Line (CMD)



\## 📂 Project Structure

\- Samples → Contains phishing email examples  

\- Reports → Contains analysis reports  

\- Python-Scripts → Automation scripts
